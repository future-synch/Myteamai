## WHAT THE SYSTEM DOES

Web-based AI assistant for estate agents. Agent types plain English.
System classifies intent, routes to correct function, calls Claude API,
syncs with HubSpot, returns structured response.
Domain-constrained to exactly 6 functions. Nothing outside these 6.

Stack: Python 3.11 + FastAPI | Vanilla JS + HTML | Claude claude-sonnet-4
       HubSpot REST v3 | JWT | Postgres | Render.com Frankfurt

---

## THE 6 BOT FUNCTIONS (TS Section 4.3)

WELCOME_CLIENT      welcome, new client, onboard        fn_generate_welcome()
REGISTER_APPLICANT  new applicant, register applicant   fn_register_applicant()
MATCH_APPLICANTS    match, find applicants, who fits    fn_match_applicants()
VALUATION_BRIEF     valuation briefing, comparables     fn_valuation_brief()
DRAFT_OUTREACH      draft, outreach, write to, contact  fn_draft_outreach()
KYC_STATUS          kyc, aml, compliance                fn_kyc_status()
UNKNOWN             none of above                       error + capability list

CRITICAL: Classifier is pure Python regex. NOT an AI call. Deterministic.
